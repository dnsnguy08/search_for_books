# **Book Search Engine**

## Description

Bootcamp - Phase 3 - Homework 21

Refactoring a REST API MERN stack application to GraphQL.

## User Story
```md
AS AN avid reader
I WANT to search for new books to read
SO THAT I can keep a list of books to purchase
```
